Integration Examples

Fireside runs an OpenAI-compatible endpoint. Any app that supports OpenAI can securely route tasks through your private models.

Open WebUI

Combine Open WebUI's massive feature set (RAG, Web Scraping, Workspaces) with Fireside's deeply private local inference engine.

  1. In Open WebUI, navigate to Admin Panel → Settings → Connections.
  2. Under the OpenAI API section, click the + button to add a custom connection.
  3. Set the API Base URL to: http://localhost:7654/v1
  4. Set the API Key to a brand new Key generated above.
  5. Click the Verify icon. You can now select your local Ollama models directly from Open WebUI's chat interface!
